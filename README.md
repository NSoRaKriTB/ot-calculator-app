# OT Calculator - เครื่องคำนวณค่าล่วงเวลา

เครื่องคำนวณค่าล่วงเวลา (OT) ตาม พ.ร.บ.คุ้มครองแรงงาน พ.ศ. 2541 สำหรับพนักงานรายเดือนและรายวัน

## Features

- คำนวณ OT 5 ประเภท ตามกฎหมายแรงงานไทย
- รองรับพนักงานรายเดือนและรายวัน
- แสดงสูตรคำนวณแบบ step-by-step
- บันทึกประวัติลง Local Storage
- ส่งออก PDF และ Excel
- Dark/Light mode
- Responsive design (mobile-first)
- SEO optimized (JSON-LD, sitemap, metadata)

## Tech Stack

- **Next.js 16** (App Router) + TypeScript
- **Tailwind CSS v4**
- **next-themes** (dark mode)
- **jsPDF** + **SheetJS** (export)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
