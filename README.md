# OT Calculator - คำนวณค่าล่วงเวลาตามกฎหมายแรงงานไทย

เครื่องมือคำนวณค่าล่วงเวลา (OT) ตาม พ.ร.บ.คุ้มครองแรงงาน พ.ศ. 2541

## Features

- คำนวณค่า OT ตามอัตรากฎหมายแรงงานไทย (1x, 1.5x, 2x, 3x)
- รองรับลูกจ้างรายเดือน รายวัน และรายชั่วโมง
- บันทึกประวัติการคำนวณ พร้อมสรุปยอดรวม
- Dark mode
- Responsive design (mobile / tablet / desktop)
- SEO optimized (meta tags, Open Graph, JSON-LD structured data)
- Accessible (ARIA labels, keyboard navigation, screen reader support)
- Print-friendly styles

## OT Rates (Thai Labor Law)

| ประเภท | อัตราทด |
|--------|--------|
| ล่วงเวลาวันทำงานปกติ | 1.5 เท่า |
| ทำงานวันหยุด (ลูกจ้างรายเดือน) | 1 เท่า |
| ทำงานวันหยุด (ลูกจ้างรายวัน) | 2 เท่า |
| ล่วงเวลาวันหยุด | 3 เท่า |

## Usage

Open `index.html` in a browser. No build step or dependencies required.

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (ES5 compatible, IIFE pattern)
- LocalStorage for data persistence
